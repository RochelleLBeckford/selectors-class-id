What are Selectors?

  • Selectors are used to determine which HTML elements(s) are styled
  • The most common type of selector is the "type selector", which selects all matching elements on the webpage for styling.

Class and Id Selectors

  • HTML elements can be selected by their HTML attributes.
  • An example that follows selects elements with a "class" of "class-name" or an "id" of "id-name":
    • Selecting an element by "class" is done using a period ".", and used to style multiple elementd with a matching "class" attribute.
    • Selecting an element by "id" is done by using a hashtag "#", and is used to style single element with a matching "id" attribute.


    • There is a way to be even more specific:
    • Instead of selecting every element with a matching class only "div" elements that either have a class of "class-name" or an "id" of "id-name".
    • This kind of selection is also known as "targeting".
